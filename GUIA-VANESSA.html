📋 Roteiro de Teste Completo

1️⃣ Dashboard Principal

  • KPIs visíveis: Total de alunos, professores, frequência geral
  • Gráfico de frequência: Por aula (barras comparativas)
  • Tendência temporal: Evolução da frequência nos últimos dias
  • Alertas: Lista de alunos em risco de evasão

2️⃣ Gestão de Alunos

  • Listar alunos: Veja os 18 alunos já cadastrados
  • Criar novo aluno:
    • Clique em "Adicionar aluno"
    • Preencha: Nome, Data de Nascimento, Nome do Responsável, Telefone
    • Salve e veja o aluno aparecer na lista
  • Buscar: Use o campo de busca para filtrar por nome
  • Filtrar: Clique em "Alunos em risco" para ver apenas os críticos
  • Ver ficha detalhada: Clique em um aluno para ver:
    • Dados pessoais e do responsável
    • Aulas matriculadas
    • Calendário de presenças
    • Notas por matéria
    • Percentual de frequência

3️⃣ Gestão de Aulas

  • Ver aulas: Acesse o menu "Aulas" (já existem 3 aulas)
  • Criar nova aula:
    • Clique em "Adicionar aula"
    • Defina: Nome da oficina, Turno (manhã/tarde/noite), Professor regente
    • Salve
  • Expandir uma aula: Clique em "Ver detalhes"
    • Veja alunos matriculados
    • Matricule novos alunos (dropdown de seleção)
    • Remova alunos (soft delete)
    • Gerencie matérias da aula

4️⃣ Relatórios

  • Panorama geral: Veja resumo de todos os alunos
  • Filtrar por aula: Use o dropdown "Aula" para focar em uma turma
  • Tabela detalhada: Veja nome, aula, nota média, frequência, aptidão, situação
  • Exportar CSV: Clique em "Exportar CSV" para baixar os dados
  • Ver ficha individual: Clique em um aluno na tabela

5️⃣ Gestão de Usuários

  • Listar usuários: Veja professores, coordenadores e admins
  • Criar novo usuário: Adicione professores ou coordenadores

✨ Principais Diferenciais do Sistema

🎯 Foco em ONG de Reforço no Contra-turno

  • Ficha independente do aluno: Não vinculada a série/ano escolar
  • Matrícula N:N: Um aluno pode fazer múltiplas aulas/oficinas
  • Oficinas temáticas: Não há conceito de "turma seriada"
  • Gestão de responsáveis: Cadastro com telefone para contato
  • Alerta de risco: Identificação automática de alunos com faltas excessivas

🔒 Segurança e Próximos Passos

⚠️ Ambiente de Demonstração

Este é um ambiente de testes com dados fictícios. As senhas são simples para facilitar a demonstração.

  • 🔐 Autenticação via Supabase Auth (senhas criptografadas)

  • 🔑 Tokens JWT para sessão segura
  • 📱 Recuperação de senha por e-mail
  • 👤 Gestão de perfis e permissões no banco
